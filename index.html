commen.tk

A very simple widget that uses Github issues and their comments to provide any page with a comment section linked with Github.

Usage

The form above will only show public repositories but feel free to use private repos as well. 

<script type='text/javascript'
    src='https://cdn.commen.tk/client.js'
    repo='[OWNER]/[REPO]'
    issue-param='title'
    theme='light'
    page-size='10'
    async
></script>
